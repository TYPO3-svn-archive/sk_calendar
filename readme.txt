README SNAPSHOT SKCALENDAR

Was Ihr hier installiert ist ein "Snapshot" des SK-Calendar, d.h. dass noch einige Dinge zu tun sind und eventuell nicht so funktionieren wie sie sollen.

Insbesondere steht auf meiner Liste:

- Beschränkung der Icons der Kategorien auf 10x10 pixel sowie .gif oder .jpg (Vermutlich $tca)
- Colorpicker zeigt noch keine Colors
- Zugriffsverwaltung wird noch ignoriert
- Schönere Icons
- TS für Währung
- Frontendplugin (das Vorhandene ist nur zum debuggen geeignet
- Templatesystem für Frontendplugin
- Anzahl Termine > 100 noch ein Problem
- Anzahl Termine wird nicht angezeigt
- Alte Termine werden noch nicht gefiltert
- Exeptions funktionieren, sind aber schlecht administrierbar.

Das was funktioniert gibt euch aber einen Eindruck zur Funktionsweise und Administration des Kalenders.
