HTML Template for sk_calendar Please notice when using markers that there are 3 different kinds Labelmarkers follow the form ###LABEL_xxxxxx#### where xxxxxx resembles a Label from pi1/locallang.php. They will be rendered taking account of the choosen language! Fieldmarkers are regular markers that render fields from the database. They consist of a letter for the table (e for event, c for category, t for targetgroups and l for location) followed by a '_' and the fieldname Example: ###e_title### gives you the eventtitle whereas ###c_color### renders the Color of the corresponding category. You can define your own PHP-Functions by extending class.tx_skcalendar_defaultTF.php. Using ###myfunct### will try to call a function templatefunc_myfunct() if it isn't there nothing happens defining a function templatefunc_e_title() would override the return of just the eventtitle but output whatever your function returns. See class.tx_skcalendar_defaultTF.php for examples. Put your override-class in uploads/sk_calendar or somewhere in fileadmin to not loose your function after updates. Have fun and return your comments to: skcalendar@sitekick.de.
